iOS App Submission Nightmare

bluetooth-peripheral

  1. the bluetooth correlative code no longer be used after redesgin app, you should del both code and:
  2. Targets -> Capabilities -> bluetooth

com.apple.security.application-groups

  1. close app group
  2. del *.entitlements
  3. grep -R --include="*.*" ".entitlements" . you’ll find hidden entitlements in Xcode -> imeican -> build setting
