mac

rMBP ME865 8g 256g
淘宝，港行，已入手三天

入手注意

包装盒/机箱/系统 ID 是否一致

三码是否合一，不一样肯定是拆封的

检测电源循环次数

一般 5 次以内，越少越好（我的是两次）

官网查询保修日期

这里（能看出卖家购买日期）

千万不要用任何优化

特别是刚 Windows 转过来的携带清理强迫症的童鞋
我早就知道 mac 不需要清理，而且是新本本，但看到号称最好的磁盘清理——CleanMyMac, 还是手贱了

清理完过了一会儿后，发现温度 90 多，还查了下 mac 散热问题
后来发现是有个进程狂飙 CPU
然后发现 Dock 许多图标变成透明的了（有字，可以点击，重启也无法解决）
一 Google，很多人居然重装了系统

而罪魁祸首居然是，CleanMyMac
血的教训，真是*不作死就不会死！！！ *

图标透明的解决办法：

Dock 上面的点一下就可以恢复；还不行就先移出，再拖进去
