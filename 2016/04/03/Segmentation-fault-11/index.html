Segmentation fault: 11

Artwalk's Blog
2016-04-03
Bug, Swift, Xcode
  • Mac OS X 10.11.4
  • Xcode 7.3
  • Swift 2.2

error: cannot convert value of type '[(String, Int)]' to expected argument type '[(name: String, age: Int)]?'

So, I renamed the tuple list type to [(name: String, age: Int)]

Everything is fine? right?

But when I Archived the project in Fastest, Aggressive Optimizations [-Ofast] and Fast, Whole Module Optimization [-O -whole-module-optimization]

Command failed due to signal: Segmentation fault: 11

The error message is irrelevant and helpless

I checked out commit by commit, and finally found that it was the name of tuple cause the error.
