Pull to Refresh UIViewController in iOS

Artwalk's Blog
2014-12-01
Programming

If you force to add a UIRefreshControl to a UIViewController, you will get an error:
UIRefreshControl may only be managed by a UITableViewController

So, here is the trick

Code

1 init a tableView in your UIViewController

2 add refreshControl to tableView, not view

3 refreshView function

If you like to use Storyboard like me

  1. Storyboard -> UITableViewController -> enable Refreshing
  2. Then you can Ctrl drag just like UIButton
