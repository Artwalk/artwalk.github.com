Artwalk's Blog
2014-06-20
Programming

xxx.xcodeproj/project.xcworkspace/xcuserdata/xxx.xcuserdatad/UserInterfaceState.xcuserstate

I was annoyed by this file.

It seems xCode update this file almost every second.
So, everytime I wanted to merge two heads, I would get a yellow conflict notification with this advantage function…

All in all, I forgot to set a global gitignore files.

This file is in your home directory, which means ~/.gitignore.

And, I found a very good website

http://www.gitignore.io/

According to your Operating System, IDE, or Programming Language, it can generate a ignore list.

Here is Xcode’s:

Do you see that *.xcuserstate above?

And your can also install it on your system by command line.

Then, you can list it by gi xcode or append it by gi xcode >> ~/.gitignore
