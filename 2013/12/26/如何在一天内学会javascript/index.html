如何半天内学会 JavaScript

LeiHao's Blog
2013-12-26
JavaScript, Programming

好吧，我只是初步的把语法糖熟悉了下而已

需要：

  • 另一门语言基础（C/C++ Java Python）
  • www.codecademy.com

解释：

  • 没语言基础的话，嗯…半天内估计…当然，这东西也是看天赋的，男默女泪
  • codecademy 这个网站好好，像打游戏一样，实时性+关卡，比看书效果好多了

数了下，13 个番茄钟通关，折合下来也就 5 小时多一点
当然整个过程用了 3 天（我实在忍不住去打魔兽啊啊啊啊啊…）
码代码的时候感觉就是在码 C+Python 嘛
码了半天才发现 JS 有’;’分号之争
我才懒得打分号呢

原来我看 HTML 源码，感觉好复杂的说，就没兴趣自习看
谁知那是 HTML CSS JS 混合在一起写的
分开来看真是简单到爆了
JS 也就简单了解下，打算好了，以后统统用 CoffeeScript
